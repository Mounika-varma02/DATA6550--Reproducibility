# Reproducibility Report: Climate Change Discussions on Reddit

## Project Overview

This project replicates the findings of a research paper examining climate-related discourse on Reddit from 2005 to 2021.
We used the authors' GitHub repository and ran the notebook in Google Colab to see if we could get the same results.

## Methodology

* Used the dataset that was shared through Google Drive links in the notebook
* Ran the notebook in Google Colab
* Made figures and compared them to the original paper

## Results

The replicated results closely resemble the original study. We were able to successfully replicate:

* Changes in sentiment over time
* The number of posts about climate change
* Analysis of feelings based on subreddits
* A comparison of Google Trends
